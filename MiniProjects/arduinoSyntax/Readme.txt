The attended project will be to use an aurdueno board with a camera module that will send information via wifi to a website (iot application)
either a phone application and/or desktop application will be used to recieve and send back information onto the ardurno board.
the ardurno board has 20 leds that can used as an idle, on, and off state.

the hardware require will be an ardurno uno 4wifi board, camera module, battery pack. a device that connect to the server.
